# SQL Data Warehouse Project

## Overview

This project demonstrates the implementation of a SQL Data Warehouse using a multi-layer architecture. The project transforms raw data into structured, analytics-ready data through the Bronze, Silver, and Gold layers.

## Objectives

- Build a SQL Data Warehouse from raw datasets.
- Clean and transform data using SQL.
- Organize data into Bronze, Silver, and Gold layers.
- Validate data quality using SQL queries.

## Tools Used

- Microsoft SQL Server
- SQL

## Project Structure
```
SQL-Data-Warehouse-Project/
│── datasets/
│── scripts/
│   ├── bronze/
│   ├── silver/
│   └── gold/
│── README.md
```

## Workflow

1. Load raw datasets into the Bronze layer.
2. Clean and transform the data in the Silver layer.
3. Create analytics-ready tables in the Gold layer.
4. Perform data quality checks using SQL queries.
